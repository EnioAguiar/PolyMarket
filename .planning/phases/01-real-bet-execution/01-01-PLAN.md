// ... tempClient uses POLY_1271 ...

After (EOA mode):
```typescript
// EOA mode: funder = signer = EOA address
const eoaAddress = account.address;

  host,
  chain,
  signer: walletClient,
  signatureType: SignatureTypeV2.EOA,
  funderAddress: eoaAddress,
});
// main client with L2 auth
clobClient = new ClobClient({
  host,
  chain,
  signer: walletClient,
  creds,
  signatureType: SignatureTypeV2.EOA,
  funderAddress: eoaAddress,
});
```
